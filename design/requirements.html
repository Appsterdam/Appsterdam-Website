User stories

Your listing

# e i s
As a member I would like to authenticate with my Twitter account so that I can manage my listing. 1 3 1 done
As a member I would like to add my listing so that I’m listed. 2 1 1 done
As a member I would like to be able to edit my listing so that I can keep it current. 3 1 2 done
As a member I would like to be able to delete my listing so that I can remove myself if I want to. 4 1 2 done
As a member I would like the app to get my Twitter ID, name, username, picture, location, website url, and bio so that I don’t have to enter these. 5 1 1 done
See http://dev.twitter.com/doc/get/users/show
As a member I would like the app to keep the information it got from Twitter updated so that changes I make on Twitter get propagated to my listing. 6 2 2 done

Browse listings

# e i s
As a visitor I would like to see an overview of membership listings so that I can get an impression of what the movement is about. 7 1 1 done
As a visitor I would like to limit the listings overview to specific attributes so that I only see what I’m interested in. 8 2 2 done
As a visitor I would like to be able to search membership listings so that I can find a specific one. 9 2 3 todo
As a member I would like the default listing to be fully randomized on each page load so that nobody gets an unfair advantage. 10 1 2 done
As a visitor I don’t want to have to click pagination links so that I don’t get annoyed. 11 1 3 done
Additional batches should load automatically when you scroll down.

Spam management

# e i s
As a visitor I would like to be able to report a membership listing or classified ad as spam so that an administrator can remove it. 20 1 3 partially
Implemented for membership listings.
As an administrator I would like to see an overview of items marked as spam so that I can remove these. 21 1 3 partially
Implemented for membership listings.
As an administrator I would like to be able to remove an item marked as spam so that I can keep the overviews free of spam. 22 1 3 partially
In the case of a membership listing, the member should not be deleted but flagged as such, otherwise he/she can just signup again. Implemented for membership listings.
As an administrator I would like to be able to “unmark” an item marked as spam in case it’s legitimate after all. 23 1 3 partially
Implemented for membership listings.
